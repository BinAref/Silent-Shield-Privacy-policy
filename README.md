# Privacy Policy Website for SilentShield

This folder contains a complete privacy policy website for the SilentShield app, designed to meet Google Play Store requirements.

## Features

### 🌐 Bilingual Support
- **English** and **Arabic** language support
- Automatic language detection based on browser settings
- RTL (Right-to-Left) layout support for Arabic
- Proper font loading for both languages

### 🌙 Dark Theme Support
- Light and dark theme toggle
- Automatic system theme detection
- Smooth theme transitions
- Persistent theme preferences

### 📱 Responsive Design
- Mobile-first approach
- Tablet and desktop optimized
- Print-friendly styles
- Accessibility compliant

### ♿ Accessibility Features
- Screen reader support
- Keyboard navigation
- High contrast mode support
- ARIA labels and announcements
- Focus management

## Files Structure

```
privacy_policy/
├── index.html          # Main HTML file

```

## Usage

1. **Host the files** on any web server or static hosting service
2. **Update the contact email** in both English and Arabic sections
3. **Customize the content** as needed for your specific app
4. **Use the URL** in your Google Play Store listing

## Hosting Options

### Free Hosting Services
- **GitHub Pages**: Upload to a GitHub repository and enable Pages
- **Netlify**: Drag and drop the folder to deploy
- **Vercel**: Connect your repository for automatic deployment
- **Firebase Hosting**: Use Firebase CLI to deploy

### Example URLs
- `https://yourusername.github.io/silentshield-privacy/`
- `https://silentshield-privacy.netlify.app/`
- `https://silentshield-privacy.vercel.app/`

## Customization

### Updating Content
1. Edit the English content in the `#english-content` div
2. Edit the Arabic content in the `#arabic-content` div
3. Update contact information in both languages

### Styling
- Modify CSS variables in `:root` for light theme colors
- Modify CSS variables in `[data-theme="dark"]` for dark theme colors
- Adjust fonts, spacing, and layout as needed

### Functionality
- Add more languages by extending the JavaScript
- Customize theme colors and behavior
- Add analytics or tracking if needed

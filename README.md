# Silent Shield — Privacy Policy Page

Hosted via GitHub Pages for the Silent Shield Android app (`com.silentshield.app`).

## Languages supported
Arabic · English · German · Spanish · French · Turkish · Chinese

## Files
- `index.html` — main page (all language content inline)
- `styles.css` — dark theme matching the app
- `script.js` — language detection and switching

## Usage
The page auto-detects the user's browser language on first visit.
Language selection is persisted via `localStorage`.
